## bluesky_profile
Look up user profiles on Bluesky.

**Tier:** All users (free, open protocol)

**Arguments:**
- **action** (string): "me" (default) or "lookup"
- **handle** (string): Bluesky handle for lookup (e.g., "user.bsky.social")

~~~json
{"action": "me"}
~~~
~~~json
{"action": "lookup", "handle": "user.bsky.social"}
~~~

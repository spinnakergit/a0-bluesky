# Bluesky Research Skill

Search and analyze content on Bluesky.

## Triggers
"search bluesky for", "find posts about", "what are people saying on bluesky"

## Steps
1. Use `bluesky_search` with action "posts" to find relevant content
2. Analyze the results for themes, sentiment, and key voices
3. Optionally use `bluesky_profile` to look up key authors
4. Summarize findings for the user

## Guidelines
- Try both "latest" and "top" sort orders for comprehensive results
- Use `bluesky_search` with action "users" to find key accounts in a space
- Cross-reference with `bluesky_read` action "user_posts" for deeper analysis
- Present findings as a structured summary

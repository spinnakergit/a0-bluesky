# Bluesky Thread Skill

Create multi-post threads on Bluesky.

## Triggers
"create a bluesky thread", "thread on bluesky", "turn this into a bluesky thread"

## Steps
1. Break the content into logical chunks of 300 graphemes or less
2. Ensure each post stands on its own while contributing to the narrative
3. Use the `bluesky_thread` tool with posts separated by `---`
4. Report the result including the first post's URI

## Guidelines
- Each post should be under 300 graphemes
- Start with a hook — the first post should grab attention
- Number posts if appropriate (e.g., "1/5: ...")
- End with a conclusion or call to action
- Maximum 25 posts per thread
